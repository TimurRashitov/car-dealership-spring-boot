Запросы:

GET
Для чего: получить данные
Safe(безопасность): да
Идемпотентность(повторяемость): да, повтор дает один и тот же результат
Кэшируемость: да
Не использовать: для создания или изменения данных

POST
Для чего: чтобы что-то создать новый ресурс
Тело запроса: JSON
Safe: нет
Идемпотентность: нет, повтор запроса создает дубликат
Кэшируемость: нет
Не использовать: получить данные

PUT
Для чего: полная замена ресурса
Тело запроса: JSON
Safe: нет
Идемпотентность: да
Кэшируемость: нет
Не использовать: если меняется только часть полей или нет возможности отправить объект целиком

PATCH
Для чего: частичное обновление
Тело запроса: JSON
Safe: нет
Идемпотентность: да
Кэшируемость: нет

DELETE
Для чего: удаление ресурса
Тело запроса: JSON
Safe: нет
Идемпотентность: да
Кэшируемость: нет
Не использовать: если запрещено физическое удаление данных

HTTP-статусы:
200 OK успешное чтение или обновление ресурса
201 Created успешное создание ресурса (POST)
204 No Content успешное обновление без тела
400 Bad Request невалидные данные (валидация DTO)
404 Not Found ресурс не найден
409 Conflict конфликт состояний
500 Internal Srver Error - неожиданная ошибка

GET: /api/cars      выдает список автомобилей
GET: /api/cars/11   выдает авто по ID
