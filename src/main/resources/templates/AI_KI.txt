Схема интеграции:

Клиент(любое устройство посылающее http Запросы) --> spring controller --> service --> HTTP Client --> AI API

Ответ AI API (JSON) --> Backend формирует ответ --> клиент.

Запрос: модель,
        роль(пример: тык как Java-разработчик проверь),
        вопрос.

1) Нельзя хранить ключи в программном коде.
2) ИИ(искусственный интеллект) может не ответить, может вернуть 404 или что-другое, токены  могут закончиться, запрос может зависнуть, нужно интегрировать логгеры чтобы анализировать

Нельзя применять ИИ в юридических консультациях, медицинских диагнозах, критические расчеты, финансовые транзакции.
